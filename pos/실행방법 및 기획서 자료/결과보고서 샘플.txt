프로젝트 개요 
	기획 의도 : 간편한 UI로 상품 구매 및 판매가 가능한 프로그램을 만들고자 하였음 
	주요 고객 : 상품 구매를 희망하는 소규모 고객들에게 배포
	주요 기능 : 주요기능.png 사진 첨부

	추진 단계 : 
		1. 협업도구 설정(깃허브)
		2. 기능 기획(관리자, 일반계정별 다른 기능), 구매,환불,회원관리,물건 추가/삭제
		3. 포토샵으로 UI 설계
		4. 텝별로 기능 구현
		5. 필요한 추가 기능 확인 및 구현(사용통계, 차트, 회원정보, 등급별 할인율)
		6. 유효성 테스트(중복확인, 특수문자, 널문자)
		7. 최종 테스트
		8. 결과보고서 작성

	프로젝트 구현 : 해당 구현 화면 이미지 첨부 + 설명
